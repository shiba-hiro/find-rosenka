# find-rosenka

A sample project to use puppeteer to download PDFs from [路線価図](https://www.rosenka.nta.go.jp/).

## Usage

```shell
git clone git@github.com:shiba-hiro/find-rosenka.git
cd find-rosenka
yarn install
yarn start
```

Then you can find PDFs in the `output` folder.

## Debug

```shell
yarn debug
```
